# PixVault — 本地图片画廊浏览器

浏览、筛选、收藏、标记不喜欢、幻灯片播放、顺序/随机模式。

## 启动

双击 `pix-vault/启动.bat`，或命令行：

```bash
cd pix-vault && python run.py
```

- 电脑：`http://localhost:8720`
- 手机（同 WiFi）：`http://192.168.31.195:8720`

## 功能

| 功能 | 说明 |
|------|------|
| 🔍 浏览 | 随机/顺序双模式，← → 历史导航 |
| 🏷️ 筛选 | 19 种标签（NSFW/正常/画风/人数/身材/构图/视角/模型） |
| ♥ 收藏 | 空格收藏/取消，多图网格浏览 |
| 💔 不喜欢 | D 标记，批量删除磁盘原文件 |
| ▶ 播放 | 幻灯片自动切换，可调间隔（3s-60s） |
| ⊞ 网格 | 缩略图画廊，点击放大，✕ 返回 |

## 快捷键

`Q` 随机 · `Space` 收藏 · `D` 不喜欢 · `← →` 导航 · `S` 播放 · `M` 模式 · `G` 网格 · `R` 刷新
